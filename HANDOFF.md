# КемСтать — Руководство по проекту

## Что это

Платформа профориентации для школьников 13–17 лет. Пользователь проходит тест RIASEC (36 вопросов), получает AI-анализ своего профиля через ChatGPT и пробует профессии через мини-игры.

Живой сайт: **career-platform-snowy.vercel.app**  
Репозиторий: **github.com/moestro-855/career-platform**

## Как работает флоу

```
/ (лендинг)
    ↓ кнопка "Пройти тест"
/test
    → NameStep: ввод имени + возраста
    → QuestionCard × 36: вопросы RIASEC (Да/Может быть/Нет)
    → calculateScores() → сохранить в localStorage
    ↓ редирект
/results
    → читает localStorage
    → POST /api/analyze → OpenAI gpt-4o-mini → 4 профессии + сильные стороны
    → показывает RIASEC-бары, рекомендации, EmailCapture
    → кнопки "Играть" → /game/[id]
/game/[id]
    → 5 сценариев: ситуация → 4 варианта → фидбек → следующий
    → итоговый экран с процентом совпадения
```

## Переменные окружения

Создать файл `.env.local` в корне проекта:

```
OPENAI_API_KEY=sk-...        # platform.openai.com — для AI-анализа
RESEND_API_KEY=re_...        # resend.com — для отправки email
```

В Vercel добавить те же переменные: **Settings → Environment Variables**.

- **Страница 404** — сейчас дефолтная Next.js, нужна брендированная

### 📝 Тест RIASEC

- **Качество вопросов** — текущие 36 вопросов написаны быстро. Нужен психолог-методолог чтобы переработать формулировки по стандарту Holland (1994)
- **Шкала ответов** — сейчас 3 варианта (0/1/2 балла). Полноценный RIASEC использует 5-балльную шкалу Ликерта
- **Нормирование** — добавить сравнение с нормативной выборкой сверстников
- **Адаптация для возраста** — вопросы для 13-летних и 17-летних должны отличаться
- **Порядок вопросов** — перемешать типы случайно, а не блоками по 6

### 💼 Профессии

- Сейчас в результатах 4 профессии от AI — они каждый раз разные и не из фиксированного списка
- Нужно создать `lib/professions.ts` с 30–50 профессиями: фото/иконка, описание, зарплата, что нужно изучать, ВУЗы
- Добавить страницу `/professions/[id]` — карточка каждой профессии
- Связать AI-рекомендации с реальными профессиями из базы

### 🎮 Мини-игры

- **Текущий формат** — текстовые сценарии с вариантами ответа (по сути второй тест)
- **Нужно** — настоящие интерактивные игры:
  - **Программист**: строки кода на экране, найди баг кликом, таймер 30 сек
  - **Дизайнер**: выбери лучший макет из 3, объяснение почему
  - **Врач**: симптомы появляются по одному, поставь диагноз до таймера
  - **Предприниматель**: управляй кофейней 5 дней — цена, реклама, меню
- Оценка: ~1 день на все 4 игры в формате Framer Motion + таймер
